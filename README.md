# 🐶 Dog Age Calculator
Welcome to the Dog Converter! This project provides tools to convert various dog-related measurements and data. 🐾

**📅 Calculate dog age in human years**

**⚛️ Built with**
* TypeScript: For strong typing and modern JavaScript features.
* React: For building a dynamic and responsive user interface.

**📦 Installation**
* Fork the repository 

* Install dependencies
```bash 
npm install 
```
* run on local host to visually see it 
```bash 
npm run dev
```
